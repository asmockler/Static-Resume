Fay.js

An animated SVG icon library for
discerning developers

Fay is an easy-to-implement animated icon library built with flexibility in mind.

Icons are added to any element simply by adding classes such as .fay-menu or .fay-chevron.

Fay icons are drawn as svg in the browser - this means that they will never be pixelated and are infinitely scalable.

Fay is built with the wonderful Raphael.js library. Icons will render and animate beautifully in tons of browsers.

Starting Fay Up

Fay.init({selector});

Initialize Fay by declaring .init() in your JavaScript file.

Fay.init("selector") accepts and optional selector as a string (i.e. div, .whatever-class or #whatever-id) to draw Fay icons within only 1 section of a page. Useful for apps with multiple widgets loaded asynchronously.

Alternately, you can make Fay go away by calling Fay.destroy() with an optional selector.

Global Attributes

  • data-fay-scale - Default is 2x the page <p> font-size.
  • data-fay-size - Overrides automatic sizing. An absolute size for an icon in px.
  • data-fay-stroke - Override for default/automatic stroke-width scaling.
  • data-fay-color - Overrides default color (icons default to color of <p> element.
  • data-fay-speed - Overrides default animation speed. Integer in miliseconds.
  • data-fay-opacity - sets opacity of given icon (from 0 to 1)

A Note About Colors

Fay's color will be automatically set to whatever is decalred for the <p> element. Colors can be changed via css or data-attribute.

Via CSS

Change the color of any of Fay's classes to flobally change color for the elements.

.fay-menu-spin{ color: 'red'; }

Alternately, use CSS3 [class*=fay-] to change the color of all icons.

Via data-attribute

Use the data-fay-color attribute to change colors of individual elements.

A Note on Sizing

By default, icons are sized to 2x your page's <p>, rounded to the nearest 5 to maximize performance. This can be overridden either with data-fay-scale or data-fay-size.

Via data-fay-scale

Scale is a relative measure based on the page's font-size for the <p> element. Accepts integers. For example, data-fay-scale=4 would size an icon to 4x the page's font-size.

Via data-fay-size

Size is an absolute scale in px for icon size. Accepts integers.
